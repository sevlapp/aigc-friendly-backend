GraphQL 端点

▼
POST http://localhost:3000/graphql

使用 POST 请求发送 GraphQL 查询，Content-Type 为 application/json

// 请求示例 { "query": "{ posts { id title } }", "variables": {} }

查询接口 (Queries)

▼

Query 获取文章详情

通过文章ID获取文章详情

post(id: Int!): Post

示例查询

Query 通过 Slug 获取文章

通过文章别名获取文章详情

postBySlug(slug: String!): Post

示例查询

Query 文章列表

分页获取文章列表，支持分类、标签、搜索过滤

posts(
  page: Int
  pageSize: Int
  categoryId: Int
  tagId: Int
  status: PostStatus
  search: String
  sortBy: String
  sortOrder: String
): PostListResult!

示例查询

Query 置顶文章

获取所有置顶文章

stickyPosts: [Post!]!

示例查询

Query 上一篇/下一篇

获取当前文章的上一篇或下一篇文章

nextPost(currentId: Int!): Post
previousPost(currentId: Int!): Post

示例查询

Query 分类列表

获取所有文章分类

categories: [Category!]!
category(slug: String!): Category

示例查询

Query 标签列表

获取所有文章标签

tags: [Category!]!
tag(slug: String!): Comment

示例查询

Query 文章评论

获取某篇文章的所有评论（支持楼中楼嵌套）

postComments(postId: Int!): [Comment!]!

示例查询

Query 博客统计

获取博客基础统计数据

blogStats: BlogStats!

示例查询

Query 时间归档

按年月统计文章数量

archiveStats: [ArchiveStats!]!

示例查询

Query 友情链接

获取所有已激活的友情链接

links: [Link!]!

示例查询

Query 系统配置

获取博客系统配置

config: [Config!]!

示例查询

变更接口 (Mutations)

▼

Mutation 创建文章

创建新的博客文章

createPost(input: CreatePostInput!): Post

示例 Mutation

Mutation 更新文章

更新已有文章

updatePost(input: UpdatePostInput!): Post

示例 Mutation

Mutation 删除文章

软删除文章（放入回收站）

deletePost(id: Int!): Boolean!

示例 Mutation

Mutation 增加阅读量

文章阅读数+1

incrementPostView(postId: Int!): Boolean!

示例 Mutation

Mutation 文章点赞

文章点赞数+1

incrementPostLike(postId: Int!): Boolean!

示例 Mutation

Mutation 创建评论

对文章发表评论

createComment(input: CreateCommentInput!): Comment

示例 Mutation

Mutation 更新评论

更新评论内容或状态

updateComment(input: UpdateCommentInput!): Comment

示例 Mutation

Mutation 审核评论

通过评论审核

approveComment(id: Int!): Boolean!

示例 Mutation

Mutation 评论点赞

评论点赞数+1

incrementCommentLike(commentId: Int!): Boolean!

示例 Mutation

Mutation 创建分类

创建新的文章分类

createCategory(input: CreateCategoryInput!): Category

示例 Mutation

Mutation 更新/删除分类

更新或删除文章分类

updateCategory(input: UpdateCategoryInput!): Category
deleteCategory(id: Int!): Boolean!

Mutation 创建/更新/删除标签

管理文章标签

createTag(input: CreateTagInput!): Category
updateTag(input: UpdateTagInput!): Category
deleteTag(id: Int!): Boolean!

Mutation 创建/更新/删除链接

管理友情链接

createLink(input: CreateLinkInput!): Link
updateLink(input: UpdateLinkInput!): Link
deleteLink(id: Int!): Boolean!

Mutation 更新配置

更新系统配置项

updateConfig(input: UpdateConfigInput!): Config

示例 Mutation

数据类型 (Types)

▼

Post - 文章

id: Int!
title: String!
slug: String!
excerpt: String
content: String!
coverImage: String
status: PostStatus!
visibility: PostVisibility!
viewCount: Int!
likeCount: Int!
isSticky: Boolean!
tags: [Tag!]!
createdAt: Date!
publishedAt: Date

Category - 分类

id: Int!
name: String!
slug: String!
description: String
parentId: Int
sortOrder: Int!
isActive: Boolean!
postCount: Int!

Tag - 标签

id: Int!
name: String!
slug: String!
description: String
postCount: Int!

Comment - 评论

id: Int!
postId: Int!
parentId: Int
authorName: String!
authorEmail: String
authorAvatar: String
content: String!
status: CommentStatus!
likeCount: Int!
replies: [Comment!]

Link - 友情链接

name: String!
url: String!
description: String
avatar: String
sortOrder: Int!
isActive: Boolean!

BlogStats - 统计数据

totalPosts: Int!
totalComments: Int!
totalViews: Int!
totalLikes: Int!

pendingComments: Int!

PostListResult - 文章列表结果

posts: [Post!]!

page: Int!
pageSize: Int!

ArchiveStats - 归档统计

year: Int!
month: Int!
count: Int!

枚举类型 (Enums)

▼

PostStatus - 文章状态

DRAFT | PUBLISHED | ARCHIVED | DELETED

PostVisibility - 文章可见性

PUBLIC | PRIVATE | PROTECTED

CommentStatus - 评论状态

PENDING | APPROVED | REJECTED | HIDDEN | DELETED

visibility: PostVisibility!

CreateCommentInput - 创建评论输入

postId: Int!

CreateCategoryInput - 创建分类输入

name: String!
